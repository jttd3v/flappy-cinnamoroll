# Proverb Ascension - Educational Skills Mapping

## Overview
Proverb Ascension develops deep comprehension, memory consolidation, and wisdom application skills through mastery-based learning mechanics.

---

## Primary Skills Developed

### 1. Reading Comprehension
- **Textual Analysis** - Understanding ancient proverbial language
- **Main Idea Extraction** - Identifying core meaning
- **Vocabulary Development** - Biblical/wisdom terminology
- **Context Interpretation** - Historical and modern relevance

### 2. Memory & Recall
- **Long-term Memory Formation** - Spaced repetition encoding
- **Retrieval Practice** - Active recall strengthening
- **Memory Consolidation** - Interval-based reinforcement
- **Verbatim vs. Gist Memory** - Balancing exact vs. understood recall

### 3. Critical Thinking
- **Paraphrasing** - Restating concepts in own words
- **Synthesis** - Combining understanding with experience
- **Application** - Connecting abstract wisdom to concrete scenarios
- **Evaluation** - Assessing quality of own understanding

### 4. Written Expression
- **Coherent Writing** - Structured explanation composition
- **Concise Communication** - Meeting word requirements meaningfully
- **Personal Reflection** - Journaling real-world connections
- **Originality** - Expressing unique interpretations

---

## Age-Appropriate Skill Targets

### Level 1-2 (Ages 6-10) - Foundation
- Basic comprehension of simple concepts
- Short phrase recall
- Guided paraphrasing with prompts
- Connecting to familiar situations (home, school)

### Level 3-4 (Ages 11-15) - Development
- Independent paraphrasing
- Multi-sentence explanations
- Personal experience connections
- Beginning abstract thinking

### Level 5-6 (Ages 16-25) - Mastery
- Deep textual analysis
- Extended written reflection
- Real-world application scenarios
- Philosophical interpretation

### Level 7 (Ages 26+) - Wisdom
- Full comprehension demonstration
- Teaching-level explanation ability
- Life experience integration
- Mentorship application

### Working Memory

- Adjusting strategies based on feedback

### Metacognition

- Recognizing shallow vs. deep comprehension
- Self-assessment accuracy

## Assessment Indicators

### Comprehension Demonstrated When:
- Paraphrase captures main idea without copying
- Personal anchor shows genuine connection
- Recall is accurate after increasing intervals
- Explanation could teach someone else

### Mastery Achieved When:
- All 5 proverbs recalled at 40-minute interval
- Written explanations meet word requirements
- Real-world applications are specific and relevant
- Player can explain without text reference

---

## Learning Transfer Goals

Players who master this game will be able to:
1. Apply spaced repetition to other learning contexts
2. Understand the value of "desirable difficulties"
3. Practice elaborative encoding naturally
4. Recognize the difference between recognition and recall
5. Value wisdom over mere information
