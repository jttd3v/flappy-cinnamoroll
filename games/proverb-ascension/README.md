# Proverb Ascension

> Wisdom must be earned, not clicked.

A mastery-based memory game where players deeply learn Proverbs 1:1-5 through understanding, application, and spaced retrieval.

## ğŸ® Game Overview

Proverb Ascension transforms scripture memorization into a cognitive challenge. Players must:
- **Read** each proverb carefully
- **Decode** its meaning through paraphrasing
- **Anchor** the wisdom to real-life experiences
- **Recall** accurately at spaced intervals

No shortcuts. No skipping. Understanding unlocks progress.

## ğŸ¯ How to Play

1. **Start** - Enter your name and age
2. **Read Phase** - Study the proverb displayed
3. **Decode Phase** - Write what it means in your own words
4. **Anchor Phase** - Describe how it applies to your life
5. **Recall Phase** - Type the proverb from memory
6. **Wait** - Proverb returns at intervals (2, 5, 10, 20, 40 min)
7. **Master** - Complete all intervals for each proverb
